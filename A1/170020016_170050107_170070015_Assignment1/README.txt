

Bi-LSTM:

Packages required:

nltk, torch, torchvision, matplotlib, numpy, argparse, gensim, json, sklearn, pandas(just for plotting), matplotlib (just for plotting), seaborn(just for plotting), spacy
Instructions:
1. Download word2vec embeddings from https://drive.google.com/file/d/0B7XkCwpI5KDYNlNUTTlSS21pQmM/edit
2. ! gunzip GoogleNews-vectors-negative300.bin.gz
3. keep this file in the same directory as bilstm_pos_tagger.py
4. ! python3 bilstm_pos_tagger.py


SVM & HMM:
These can be run by running `python3 solver.py --model <HMM/SVM>`
You need to download glove.6B.100d.txt in this folder to make it run.
